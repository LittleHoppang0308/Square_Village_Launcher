🛠 개선 사항
■ 사용자 추가 파일 유지 문제 일부 수정

기존 런처에서는 개인적으로 추가한
모드(Mod), 리소스팩(Resource Pack), 쉐이더(Shader) 파일이 있는 상태에서

[플레이] 버튼을 눌러 게임을 실행할 경우
일부 파일이 사라지는 현상이 발생할 수 있었습니다.

이번 업데이트를 통해
런처 실행 과정에서의 파일 동기화 및 관리 방식을 개선하여
사용자가 추가한 파일이 유지되도록 일부 패치가 적용되었습니다.

■ 런처 파일 처리 안정성 개선

런처 실행 과정에서 이루어지는
모드 및 리소스팩 파일 검사 과정의 안정성을 개선하였습니다.

이를 통해 런처 실행 시 발생할 수 있던
일부 파일 처리 오류 가능성이 완화되었습니다.
